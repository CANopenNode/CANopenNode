In order for the driver to work you need to download at least the S32 SDK from:
https://www.nxp.com/
or download our S32DS: S32 Design Studio IDE for ARM or POWERPC which already contains the SDK, from

The driver was developed and tested using S32K144EVB-Q100 under S32DS for ARM, but it should work with any of the S32 SDK supported devices and enviroments.
