Parameters

OD_EEPROMAddressAddress of OD_EEPROM structure from object dictionary.
OD_EEPROMSizeSize of OD_EEPROM structure from object dictionary.
OD_ROMAddressAddress of OD_ROM structure from object dictionary.
OD_ROMSizeSize of OD_ROM structure from object dictionary.
Returns
CO_ReturnError_t: CO_ERROR_NO, CO_ERROR_DATA_CORRUPT (Data in eeprom corrupt) or CO_ERROR_CRC (CRC from MBR does not match the CRC of OD_ROM block in eeprom).
