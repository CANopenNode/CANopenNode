Detailed Description

Main CANopen stack file.

It combines Object dictionary (CO_OD) and all other CANopen source files. Configuration information are read from CO_OD.h file. This file uses one CAN module. If multiple CAN modules are to be used, then this file may be customized for different CANopen configuration. (One or multiple CANopen device on one or multiple CAN modules.)
